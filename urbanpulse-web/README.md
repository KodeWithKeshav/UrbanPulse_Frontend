# UrbanPulse Frontend

A modern civic complaint platform built with React, Vite, and Tailwind CSS.

## Features
- AI-powered image validation via Roboflow
- Interactive heatmap analytics with Leaflet
- Multi-step complaint submission wizard
- Admin dashboard with priority queuing
- Real-time chatbot assistance
- Geolocation-based privacy controls
- Community upvoting system
