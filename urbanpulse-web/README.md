# UrbanPulse Frontend

A modern civic complaint platform built with React, Vite, and Tailwind CSS.

## Features
- AI-powered image validation
- Interactive heatmap analytics
- Multi-step complaint submission wizard
- Admin dashboard with priority queuing
- Real-time chatbot assistance

## Setup
```bash
npm install
npm run dev
```

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Leaflet + react-leaflet
- Chart.js
- Framer Motion
